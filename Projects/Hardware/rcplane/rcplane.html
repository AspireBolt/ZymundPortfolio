Project Overview

This project focuses on designing and building a custom RC plane and its controller using Arduino technology. The aircraft is powered by a brushless motor, controlled through an ESC, and equipped with servo motors for precise control of flight surfaces. Essential components such as a lithium battery with XT60 connectors, a charger, and bullet banana connectors ensure reliable power delivery. The airframe is constructed from lightweight foam board with landing gear. Bluetooth communication is enabled via an HC-05 module, allowing the RC plane to be remotely operated using a smartphone. By combining accessible components with Arduino programming, this project demonstrates how to make a flyable, phone-controlled RC plane from scratch, offering both a practical learning experience and an engaging hands-on application of electronics, mechanics, and aerodynamics.

Key Features
